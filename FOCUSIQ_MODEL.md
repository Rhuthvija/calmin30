# 🤖 FocusIQ – Predicting Focus & Procrastination Using Machine Learning

## Overview

FocusIQ is the machine learning component behind the **CalmIn30** platform.

The idea for this project came from a simple question:

> *Can everyday habits such as sleep, phone usage, stress, and study time be used to estimate how focused a student is likely to be?*

To explore this, I built a machine learning pipeline that predicts a student's **Focus Score** and estimates their **Procrastination Risk** based on behavioural and lifestyle data.

The model is intended to provide meaningful insights that can encourage healthier study habits rather than simply measuring productivity.

## Problem Statement

Many students struggle with maintaining focus due to multiple factors that interact with one another.

Instead of looking at only one metric, FocusIQ considers several daily habits together to identify patterns that influence concentration and productivity.

The goal is to help students better understand the relationship between their routines and their ability to focus.

---

## Input Features

The model uses the following features for prediction:

- Sleep Hours
- Study Hours
- Phone Usage
- Stress Level
- Motivation Level
- Exercise Duration

These features were selected because they have a direct influence on focus, productivity, and academic performance.

The project follows a standard machine learning workflow:

7. Prediction Generation

---

## Models Used

### Random Forest Regressor

Used to predict the student's **Focus Score**.

---

### Support Vector Machine (SVM)

Used to classify the user's **Procrastination Risk** as Low, Medium, or High.

---

### K-Means Clustering

Used to identify groups of students with similar behavioural patterns, enabling more personalized insights.

## Integration with CalmIn30

FocusIQ serves as the prediction engine for the CalmIn30 web application.

The web application collects user inputs through an interactive interface and passes them to the machine learning model, which generates predictions and personalized productivity insights.

This separation keeps the machine learning pipeline independent while enabling integration into a user-friendly application.

## Personalized Suggestions

Instead of only predicting a score, FocusIQ generates simple recommendations based on the user's inputs.

Example:

> 🏃 **Add some physical activity to your daily routine.**

Future recommendations may include:

- 😴 Increase sleep duration.
- 📱 Reduce phone usage while studying.
- 📚 Increase focused study sessions.
- 🧘 Practice mindfulness or grounding exercises.
- 🎯 Set realistic daily study goals.
- 🚶 Include regular physical activity.
- ☕ Take structured breaks using techniques such as the Pomodoro method.

The objective is to provide practical, easy-to-follow suggestions that encourage healthier habits and improved productivity rather than simply displaying prediction results.

---

## Why These Recommendations Matter

FocusIQ is designed as a **decision-support tool**, not a replacement for professional guidance.

The recommendations are intended to help users:

- Recognize lifestyle patterns affecting concentration.
- Build healthier study routines.
- Improve consistency over time.
- Reduce procrastination through small, achievable changes.
- Develop greater self-awareness regarding daily habits.
